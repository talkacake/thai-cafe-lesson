The Rules: All 4 Forms

กฎ: ทั้ง 4 รูปแบบ

Positive (+)

There is a cat.
There are some cats.

Negative (-)

There isn't a cat.
There aren't any cats.

Question (?)

Is there a cat?
Are there any cats?

Short Answer (Yes/No)

Yes, there is. / No, there isn't.
Yes, there are. / No, there aren't.
